### Автоматизировать проверки методов сервиса

Сервис доступен по следующему адресу: https://alexwohlbruck.github.io/cat-facts/docs/endpoints/facts.html

• GET /facts/random<br>
• GET /facts/:factID<br>
• GET /facts/me<br>

#### Задание: 

1. Необходимо продумать минимум 3 теста (минимум по одному тесту на метод)
2. Дополнительные тесты приветствуются, так же можно писать тесты на негативные сценарии.<br>

### Требования:
• Все тесты должны проходить<br>
• Использовать RestAssured или Feign в качестве клиента (они уже подключены в BuilderInference.gradle)
